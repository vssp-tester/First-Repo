# EnviroLab – GitHub + Vercel Setup

## 1. Create the project locally

```bash
npm create vite@latest envirolab -- --template react
cd envirolab
npm install
```

## 2. Replace files

Delete everything inside `src/` and replace with the files below.

---

## File: `src/main.jsx`

```jsx
import React from "react";
import ReactDOM from "react-dom/client";
import App from "./App";

ReactDOM.createRoot(document.getElementById("root")).render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);
```

---

## File: `src/App.jsx`

Paste the full App component code here.
The only change needed at the **top** of the file is to add:

```jsx
import React, { useState, useRef, useMemo, useEffect, memo } from "react";
```

Replace the existing line:
```jsx
import { useState, useRef, useMemo, useEffect, memo } from "react";
```

```bash

```

Open http://localhost:5173 — should look identical to the Claude artifact.

---

## 4. Push to GitHub

```bash
git init
git add .
git commit -m "Initial EnviroLab commit"
gh repo create envirolab --public --push --source=.
```

> If you don't have the GitHub CLI, create the repo manually on github.com,
> then:
> ```bash
> git remote add origin https://github.com/YOUR_USERNAME/envirolab.git
> git branch -M main
> git push -u origin main
> ```

---

## 5. Deploy with Vercel

1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Import your `envirolab` GitHub repo
3. Vercel auto-detects Vite — keep default settings:
   - **Framework**: Vite
   - **Build command**: `vite build`
   - **Output directory**: `dist`
4. Click **Deploy**

Every `git push` to `main` will trigger an automatic redeploy.
